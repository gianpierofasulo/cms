CMS (Content Management System) sviluppato con Laravel aggiornato alla Versione 11
Caratteristiche principali
<ul>
    <li>Multilingua Il CMS è completamente multilingua, si possono creare e gestire contenuti in più lingue senza duplicazioni di dati (file multilingua in JSON)</li>
    <li>Carrello e-commerce Il CMS include un sistema di carrello integrato, progettato per gestire transazioni di prodotti e servizi in maniera semplice ed efficace.</li>
    <li>Integrazione con gateway di pagamento PayPal/Stripe</li>
    <li>Interfaccia amministrativa</li>
    <li>Gestione Ruoli</li>
    <li>Gestione permessi</li>
    <li>Gestione pagine statiche</li>
    
</ul>
